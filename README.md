## 🚀 Features

- 🔐 **Login Screen** — User authentication flow
- 🏠 **Feed** — Browse posts from people you follow
- 🔍 **Search** — Discover users and content
- 🔔 **Notifications** — Real-time activity alerts
- 👤 **Profile** — View and manage your profile
- 🌙 **Dark Theme** — Sleek dark UI with pink accent colors
- 📱 **Cross-platform** — Runs on iOS, Android & Web

## 🎨 Theme

SOCIALOOP uses a custom dark theme defined in `Theme.js`:

| Token           | Color     | Usage                   |
|-----------------|-----------|-------------------------|
| `background`    | `#111827` | App background          |
| `surface`       | `#1f2937` | Cards, headers, tab bar |
| `primary`       | `#ec4899` | Buttons, active icons   |
| `secondary`     | `#db2777` | Highlights              |
| `accent`        | `#fbcfe8` | Light pink accents      |
| `text`          | `#f9fafb` | Primary text            |
| `textSecondary` | `#9ca3af` | Subtitles, captions     |

## ⚙️ Getting Started

### Prerequisites

- Node.js (v16+)
- Expo CLI installed globally
- iOS Simulator / Android Emulator **or** Expo Go app on your phone

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/socialoop.git
   cd socialoop
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the app**
   ```bash
   npm start
   ```

   Then scan the QR code with **Expo Go** on your phone, or press:
   - `a` → Open on Android Emulator
   - `i` → Open on iOS Simulator
   - `w` → Open in Web Browser

## 📜 Available Scripts

| Script            | Description                   |
|-------------------|-------------------------------|
| `npm start`       | Start Expo development server |
| `npm run android` | Launch on Android emulator    |
| `npm run ios`     | Launch on iOS simulator       |
| `npm run web`     | Launch in web browser         |
